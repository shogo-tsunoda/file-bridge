Privacy Policy for File Bridge

Last updated: March 2, 2026

1. Introduction

File Bridge is a free, open-source desktop application developed by shogo-tsunoda that enables file transfers from iPhone to Windows PC over a local Wi-Fi network. This Privacy Policy explains how File Bridge handles your data.

2. Data Collection

File Bridge does NOT collect, store, transmit, or share any personal data. Specifically:

- No personal information is collected (name, email, phone number, etc.)
- No usage data or analytics are collected
- No telemetry or crash reports are sent
- No cookies or tracking technologies are used
- No account creation or login is required
- No advertising or ad-related data is processed

3. How File Bridge Works

File Bridge operates entirely on your local Wi-Fi network. When you use the app:

- A local HTTP server runs on your Windows PC, accessible only within your local network.
- Your iPhone connects to this local server via your Wi-Fi network to upload files.
- Files are transferred directly from your iPhone to your PC and saved to a folder you choose.
- No data is transmitted to any external server or third-party service.
- No internet connection is required for file transfers.

4. Network Communication

All network communication occurs exclusively within your local Wi-Fi network. File Bridge does not connect to, communicate with, or send any data to external servers, cloud services, or any other remote endpoints.

5. File Storage

Files transferred using File Bridge are saved only to the local folder you specify on your Windows PC. The application does not copy, upload, or back up your files to any external location.

6. Third-Party Services

File Bridge does not use any third-party services, SDKs, analytics tools, or advertising frameworks.

7. Children's Privacy

File Bridge does not collect any personal information from anyone, including children under the age of 13.
